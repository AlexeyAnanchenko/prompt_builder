# Prompt Builder - Streamlit версия

Это веб-версия приложения Prompt Builder, переписанная с Tkinter на Streamlit.

## Запуск приложения

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
streamlit run app.py
```

Или через Python модуль:
```bash
python -m streamlit run app.py
```

3. Откройте браузер по адресу: `http://localhost:8501`

## Особенности реализации

### Архитектура
- **app.py** - основной файл приложения Streamlit
- Сохранение состояния через `st.session_state`
- Кэширование ресурсов через `@st.cache_resource` и `@st.cache_data`

### Основные компоненты
1. **Верхняя панель управления**:
   - Переключатель показа/скрытия системного промпта
   - Выбор namespace
   - Кнопка обновления векторной БД

2. **Системный промпт**:
   - Условное отображение
   - Кнопки очистки и копирования

3. **Основной контент**:
   - Левая колонка: "Мой запрос"
   - Правая колонка: "Готовый промпт"
   - Кнопки управления для каждого поля

4. **Нижняя панель**:
   - Основная кнопка "Сгенерировать"

### Интеграция с бэкендом
- Используются мок-объекты для демонстрации
- Реальные модули можно подключить, заменив мок-классы
- Кэширование предотвращает перезагрузку ресурсов

## Следующие шаги для полной интеграции

1. **Подключение реальных модулей**:
   - Заменить мок-классы на реальные импорты
   - Настроить подключение к БД
   - Интегрировать векторное хранилище

2. **Интеграция core_logic**:
   - Подключить `core_logic.get_full_context()`
   - Интегрировать `core_logic.generate_sql_inserts()`

3. **Улучшение UI**:
   - Добавить обработку ошибок
   - Реализовать прогресс-бары
   - Добавить дополнительные настройки

## Преимущества Streamlit версии

- **Доступность**: Работает в браузере
- **Простота развертывания**: Легко делиться с командой
- **Интерактивность**: Автоматическое обновление интерфейса
- **Кэширование**: Оптимизация производительности
